## الرسالة والرؤية

ديولي هي منصة ثنائية اللغة مبتكرة مصممة لتسهيل المناظرات الفكرية والمناقشات العلمية وعروض المواهب. مهمتنا هي خلق مساحة يمكن فيها تبادل الأفكار باحترام ومشاركة المعرفة واكتشاف المواهب.

### الرؤية

أن نصبح المنصة العربية الرائدة للمناظرات والحوار والمنافسات، وربط المفكرين والعلماء والموهوبين في جميع أنحاء العالم العربي وخارجه.

### القيم

- **الاحترام**: يجب أن تحافظ جميع المناقشات على الاحترام المتبادل
- **المعرفة**: تعزيز الحجج القائمة على الأدلة
- **التنوع**: الترحيب بجميع وجهات النظر والخلفيات
- **الابتكار**: الاستفادة من التكنولوجيا لتحقيق تواصل هادف

## نظرة عامة على المنصة

### ما هي ديولي؟

ديولي هي منصة قائمة على الويب تمكّن من:

1. **المناظرات المباشرة**: مناظرات فيديو في الوقت الحقيقي بين مشاركين أو أكثر
2. **المنافسات المسجلة**: مناظرات مؤرشفة متاحة للمشاهدة
3. **الأحداث المجدولة**: منافسات قادمة مع التسجيل
4. **عروض المواهب**: منافسات قائمة على الأداء
5. **المناقشات العلمية**: حوارات أكاديمية وبحثية

### الجمهور المستهدف

- **المتناظرون**: سياسيون، محللون، مفكرون
- **العلماء**: باحثون، أكاديميون، طلاب
- **المواهب**: شعراء، مغنون، فنانون، مؤدون
- **المشاهدون**: الجمهور العام المهتم بالمحتوى الفكري
- **المؤسسات**: وسائل الإعلام، المؤسسات التعليمية

### إحصائيات المنصة

| المقياس | الهدف |
|---------|-------|
| المستخدمون النشطون شهرياً | +100,000 |
| المنافسات المباشرة/يوم | +50 |
| المحتوى المسجل | +10,000 ساعة |
| الدول المتغطية | +25 دولة عربية |

## الميزات الأساسية

### 1. البث المباشر

**الوصف**: بث فيديو في الوقت الحقيقي للمنافسات المباشرة

**القدرات**:
- بث عبر WebRTC من نظير لنظير
- دعم متعدد المشاركين (حتى 4 موجات فيديو)
- زمن استجابة منخفض (أقل من 2 ثانية)
- بث بمعدل بت متكيف
- إمكانية التسجيل

**التنفيذ التقني**:
- خدمة P2PConnection لـ WebRTC
- خادم TURN لاجتياز NAT
- خادم إشارات لتأسيس الاتصال
- VideoCompositor لإدارة التخطيط

### 2. إدارة المنافسات

**الوصف**: إدارة كاملة لدورة حياة المنافسات

**حالات المنافسة**:
```
قيد الانتظار → مقبولة → مباشرة → مكتملة
                      ↘ ملغاة
```

**الميزات**:
- إنشاء منافسة مع القواعد والوصف
- نظام طلب الانضمام للخصم
- الجدولة مع التذكيرات
- تصنيف الفئة والفئة الفرعية
- وسم اللغة والبلد

### 3. تفاعل المستخدم

**الوصف**: ميزات المشاركة للمشاهدين

**الميزات**:
- تعليقات مباشرة أثناء المنافسات
- نظام الإعجاب/عدم الإعجاب
- تقييم بالنجوم (1-5) للمتنافسين
- نظام المتابعة للمستخدمين
- وظيفة المشاركة

### 4. نظام الإشعارات

**الوصف**: إشعارات في الوقت الحقيقي لمشاركة المستخدم

**أنواع الإشعارات**:
| النوع | الوصف |
|-------|-------|
| `طلب` | طلب انضمام جديد للمنافسة |
| `متابعة` | متابع جديد |
| `تعليق` | تعليق على منافستك |
| `تقييم` | تقييم جديد مستلم |
| `نظام` | إعلانات المنصة |
| `دعوة` | دعوة منافسة |

### 5. نظام المراسلة

**الوصف**: مراسلة خاصة بين المستخدمين

**الميزات**:
- تسليم الرسائل في الوقت الحقيقي
- إدارة المحادثات
- إيصالات القراءة
- سجل الرسائل

### 6. البحث والاكتشاف

**الوصف**: آليات اكتشاف المحتوى

**الميزات**:
- بحث شامل (مستخدمون، منافسات)
- تصفية حسب الفئة
- تصفية حسب الحالة (مباشر، مسجل، قادم)
- تصفية حسب اللغة
- تصفية حسب البلد

### 7. ملفات المستخدم

**الوصف**: ملفات مستخدم شاملة

**عناصر الملف**:
- اسم العرض والصورة الرمزية
- السيرة الذاتية والبلد
- الإحصائيات (منافسات، انتصارات، مشاهدات، تقييم)
- سجل المنافسات
- المتابعون/المتابَعون

### 8. المصادقة

**الوصف**: مصادقة مستخدم آمنة

**الطرق**:
- بريد إلكتروني/كلمة مرور مع التحقق
- Google OAuth
- Facebook OAuth
- Microsoft OAuth
- TikTok OAuth

## أدوار المستخدمين والصلاحيات

### زائر (غير مصادق)

**الصلاحيات**:
- مشاهدة المنافسات المباشرة
- مشاهدة المنافسات المسجلة
- تصفح الفئات
- البحث في المحتوى
- مشاهدة ملفات المستخدمين

**القيود**:
- لا يمكن التعليق
- لا يمكن التقييم
- لا يمكن متابعة المستخدمين
- لا يمكن إنشاء منافسات

### مستخدم (مصادق)

**الصلاحيات**:
- جميع صلاحيات الزائر
- التعليق على المنافسات
- تقييم المتنافسين
- متابعة المستخدمين
- إنشاء منافسات
- إرسال رسائل
- استلام إشعارات
- تخصيص الإعدادات

**القيود**:
- لا يمكن إدارة المحتوى
- لا يمكن الوصول لميزات المشرف

### مستخدم موثق

**صلاحيات إضافية**:
- عرض شارة التوثيق
- أولوية في نتائج البحث
- جودة بث أعلى

### مشرف

**صلاحيات إضافية**:
- إدارة المستخدمين (عرض، تعديل، حذف)
- إدارة المحتوى
- إدارة البلاغات
- الوصول للتحليلات
- تكوين المنصة

## نظام المنافسات

### دورة حياة المنافسة

```
┌─────────────┐
│   تم إنشاؤها  │
│ (قيد الانتظار) │
└──────┬──────┘
       │
       ▼
┌─────────────┐     ┌─────────────┐
│   مقبولة    │────▶│   ملغاة     │
│ (لديها خصم) │     │             │
└──────┬──────┘     └─────────────┘
       │
       ▼
┌─────────────┐
│   مباشرة    │
│  (بث حي)    │
└──────┬──────┘
       │
       ▼
┌─────────────┐
│   مكتملة    │
│  (مسجلة)    │
└─────────────┘
```

### تدفق طلب المنافسة

```
المنشئ ينشئ منافسة (قيد الانتظار)
         │
         ▼
المستخدمون يرسلون طلبات الانضمام
         │
         ▼
المنشئ يراجع الطلبات
         │
         ├──▶ قبول → المنافسة تصبح "مقبولة"
         │
         └──▶ رفض → يتم إزالة الطلب
         │
         ▼
يصل وقت الجدولة
         │
         ▼
المنافسة تصبح "مباشرة"
         │
         ▼
تنتهي المنافسة → "مكتملة"
```

### نظام التقييم

**التقييمات الفردية**:
- يمكن لكل مشاهد تقييم كل متنافس (1-5 نجوم)
- التقييمات مجهولة
- تقييم واحد لكل مستخدم لكل متنافس لكل منافسة

**متوسط التقييم**:
- يُحسب كمتوسط لجميع التقييمات
- يُعرض في ملف المستخدم
- يساهم في سمعة المستخدم

## نموذج الربح

### مصادر الإيرادات

#### 1. الإعلانات

**التنفيذ**:
- إعلانات قبل المحتوى المسجل
- إعلانات بانر أثناء البث المباشر
- منافسات برعاية

**تقاسم الإيرادات**:
- 70% لمنشئ المحتوى
- 30% للمنصة

#### 2. التبرعات

**الميزات**:
- يمكن للمشاهدين التبرع للمنشئين
- مستويات تبرع متعددة

**الميزات المقترحة**:
- مشاهدة بدون إعلانات
- بث عالي الدقة
- الوصول للمحتوى الحصري
- شارات ملف مخصصة

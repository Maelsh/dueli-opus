# User Guide - دليل المستخدم

Welcome to Dueli! This guide will help you get started with using the platform for live debates and discussions.

## Getting Started / البدء

### Creating an Account / إنشاء حساب

1. **Visit the Homepage**: Go to [dueli.pages.dev](https://dueli.pages.dev)
2. **Click "Sign Up"**: Choose your preferred registration method
3. **Choose Authentication Method**:
   - **Email Registration**: Provide email, password, and username
   - **OAuth Login**: Use Google, Facebook, Microsoft, or TikTok
4. **Verify Email**: Check your email for verification link (if using email registration)
5. **Complete Profile**: Add display name, bio, and preferences

### Setting Up Your Profile / إعداد الملف الشخصي

After registration:

1. **Go to Profile**: Click your avatar or username in the top navigation
2. **Edit Information**:
   - Display name
   - Bio/description
   - Country and language preferences
   - Profile picture (optional)
3. **Privacy Settings**: Choose profile visibility and notification preferences

## Navigating the Platform / تصفح المنصة

### Homepage / الصفحة الرئيسية

The homepage shows:
- **Featured Competitions**: Popular and live debates
- **Recent Activity**: Latest competitions and comments
- **Categories**: Browse by topic (Politics, Sports, Technology, etc.)
- **Live Now**: Currently active debates

### Main Navigation / القائمة الرئيسية

- **Home**: Main feed and featured content
- **Explore**: Browse all competitions by filters
- **Create**: Start a new debate competition
- **Profile**: Your account and settings
- **Notifications**: Messages and updates

## Creating Competitions / إنشاء منافسات

### Starting a Debate / بدء مناظرة

1. **Click "Create"**: From the main navigation
2. **Fill Competition Details**:
   - **Title**: Clear, descriptive title
   - **Description**: Detailed explanation of the debate topic
   - **Rules**: Guidelines for participants and viewers
   - **Category**: Choose relevant category and subcategory
   - **Language**: Arabic or English
   - **Schedule**: Set date and time (optional)
3. **Privacy Settings**: Choose if competition is public or private
4. **Submit**: Your competition is now live for others to join

### Competition Types / أنواع المنافسات

- **Public Competitions**: Anyone can view and join
- **Private Competitions**: Invitation-only
- **Scheduled**: Pre-planned debates
- **Live**: Immediate start debates

## Participating in Competitions / المشاركة في المنافسات

### Joining a Competition / الانضمام لمناظرة

1. **Browse Competitions**: Use Explore page or homepage
2. **Find Interesting Debate**: Read title, description, and rules
3. **Request to Join**: Click "Join" button
4. **Wait for Approval**: Creator will accept or decline your request
5. **Prepare**: Review rules and prepare your arguments

### During Live Debate / أثناء المناظرة الحية

1. **Join Live Room**: Click "Join Live" when debate starts
2. **Video Participation**: Share your video and screen if needed
3. **Real-time Chat**: Comment and discuss with viewers
4. **Rating System**: Rate participants during and after debate
5. **Follow Guidelines**: Respect debate rules and platform policies

### Viewing Competitions / مشاهدة المنافسات

- **Watch Live**: Join active debates as spectator
- **View Recorded**: Watch completed debates on YouTube
- **Comment**: Share thoughts and feedback
- **Rate**: Rate participants' performance
- **Share**: Share debates with others

## Troubleshooting / حل المشاكل

### Common Issues / المشاكل الشائعة

**Can't join live debate**
- Check internet connection
- Ensure camera/microphone permissions
- Try refreshing the page

**Notifications not working**
- Check browser notification settings
- Verify email preferences
- Clear browser cache

**Video quality issues**
- Check internet speed
- Close other applications
- Use wired connection if possible

**Language display problems**
- Clear browser cache
- Check language settings
- Try different browser

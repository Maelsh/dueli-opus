# أهداف منصة Dueli

## الأهداف الاستراتيجية

### 1. إنشاء ساحة حوار عامة
- تحويل النزاعات لحوار مفتوح
- تحويل العلوم والنظريات الصعبة لثقافة عامة
- تحويل المنافسة بين الموهوبين إلى توجيه للنفع في المكان المناسب

### 2. تحقيق الشفافية المطلقة
- كل شيء معروض علنياً (البلاغات، الإجراءات، الرواتب)
- المبلّغ والمُبلّغ عنه معروفين
- الأدوار والمسؤوليات واضحة للجميع

### 3. العدالة في التقييم
- الجمهور هو الحكم
- نظام نقاط شفاف (0-100)
- لا تلاعب ممكن

### 4. الربح العادل
- 80% من أرباح الإعلانات للمتنافسين
- 20% للإدارة والتطوير
- التوزيع بنسبة التقييم

## الأهداف التقنية

### 1. منصة مفتوحة المصدر
- الكود متاح للجميع
- إمكانية المساهمة (Contributions)
- إمكانية Self-hosting
- **لكن:** منع الاستخدام التجاري بدون إذن من Maelsh Pro

### 2. دعم عالمي
- كل اللغات مدعومة
- دعم RTL/LTR تلقائي
- توقيت محلي لكل منطقة

### 3. بث مباشر موثوق
- WebRTC للبث (P2P)
- Jitsi كـ fallback
- Cloudflare للحماية والتوزيع

### 4. قاعدة بيانات سريعة
- D1 لقاعدة البيانات
- Workers للـ API serverless
- استجابة سريعة

## الأهداف المالية

### نموذج الربح الشفاف

#### توزيع الإيرادات:
```
إيراد الإعلانات = 100%
├─ 20% → إدارة + تطوير (Maelsh Pro)
└─ 80% → المتنافسين
    ├─ حسب نسبة النقاط
    └─ (نقاط المتنافس ÷ مجموع النقاط الكلي)
```

#### الشفافية المالية:
- تتبع كل إعلان
- معرفة الربح من كل منافسة
- فواتير وإيصالات واضحة
- الاستلام متزامن مع دفع المعلنين

### طرق الدفع:
- حسابات بنكية
- بطاقات مسجلة
- منح ودعم

## الأهداف الاجتماعية

### 1. بناء مجتمع واعي
- تشجيع الحوار البناء
- رفع مستوى النقاش
- نشر الوعي بالقضايا المهمة

### 2. اكتشاف المواهب
- إعطاء فرصة للموهوبين
- تقييم عادل بدون محسوبية
- ربط المواهب بالفرص

### 3. تثقيف الجمهور
- وصول العلم للجميع
- مناقشات علمية مفتوحة
- تبسيط المعلومات المعقدة

## مؤشرات النجاح (KPIs)

### الأشهر الستة الأولى:
- 50,000 مستخدم مسجل
- 1,000 منافسة شهرياً
- متوسط مشاهدة 25 دقيقة
- نسبة تفاعل 15%
- $5,000 إيرادات شهرية

### السنة الأولى:
- 500,000 مستخدم مسجل
- 10,000 منافسة شهرياً
- متوسط مشاهدة 40 دقيقة
- نسبة تفاعل 25%
- $50,000 إيرادات شهرية

## الأولويات

### المرحلة 1: MVP (الشهر الأول)
- تسجيل دخول (OAuth)
- فتح منافسة + طلبات
- بث مباشر (WebRTC)
- تعليقات + تقييم بسيط

### المرحلة 2: التفاعل (الشهر 2-3)
- نظام سمعة كامل
- إشعارات
- بحث واكتشاف
- متابعة مستخدمين

### المرحلة 3: الربح (الشهر 4-6)
- إعلانات
- اشتراكات Pro (اختياري)
- تحليلات للمنشئين
- نظام الدفع الكامل
